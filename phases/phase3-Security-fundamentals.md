# Phase 3 — Security Fundamentals

This phase teaches the core cybersecurity concepts every SOC analyst must understand.

---

# What You Need To Learn

## 1. CIA Triad

Learn:
- Confidentiality
- Integrity
- Availability

Understand:
- Why systems need protection
- Business impact of attacks

---

## 2. Authentication & Authorization

Learn:
- Passwords
- MFA
- SSO
- OAuth
- Tokens

Understand:
- Identity attacks
- Credential theft

---

## 3. Common Attack Types

Learn:
- Phishing
- Malware
- Ransomware
- Brute force
- Password spray
- Credential stuffing

Understand:
- How attacks occur
- Indicators of compromise

---

## 4. Malware Fundamentals

Learn:
- Trojan
- Worm
- RAT
- Spyware
- Rootkits

Understand:
- Malware execution
- Persistence mechanisms

---

## 5. MITRE ATT&CK Framework

Learn:
- Tactics
- Techniques
- Procedures

Important Tactics:
- Initial Access
- Persistence
- Privilege Escalation
- Defense Evasion
- Credential Access
- Discovery
- Lateral Movement

Why It Matters:
- SOC investigations heavily use MITRE mapping

---

## 6. Indicators Of Compromise (IOCs)

Learn:
- Malicious IPs
- Hashes
- Domains
- URLs

Understand:
- IOC enrichment
- Threat intelligence

---

## 7. Incident Response Basics

Learn:
- Identification
- Containment
- Eradication
- Recovery

Understand:
- SOC escalation process
- Severity handling

---

# Recommended Labs

Practice:
- Analyze phishing emails
- Investigate suspicious IPs
- Study malware behavior
- Map attacks to MITRE

---

# Goal Of This Phase

By the end of this phase, you should:
- Understand cyber attacks
- Understand attacker behavior
- Read security alerts confidently
- Understand SOC workflows
