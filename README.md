# SOC Analyst Learning Path

## Phase 1 — IT Fundamentals

Learn:
- Networking Basics
- TCP/IP
- DNS
- HTTP/HTTPS
- Operating Systems
- Linux Basics
- Windows Event Logs
- Active Directory Basics

Tools:
- Wireshark
- Windows Event Viewer
- Linux Terminal

---

## Phase 2 — Security Fundamentals

Learn:
- CIA Triad
- Authentication
- Authorization
- MFA
- VPN
- Malware Types
- Phishing
- MITRE ATT&CK Framework

Understand:
- Initial Access
- Persistence
- Privilege Escalation
- Defense Evasion
- Lateral Movement

---

## Phase 3 — SIEM & Log Analysis

Learn:
- SIEM Fundamentals
- Log Collection
- Correlation Rules
- Alert Triage
- IOC Analysis

Platforms:
- Microsoft Sentinel
- Splunk
- QRadar
- Elastic

Log Sources:
- Windows Logs
- Azure AD Logs
- Firewall Logs
- EDR Alerts
- VPN Logs
- Email Security Logs

---

## Phase 4 — SOC Investigations

Investigate:
- Impossible Travel
- MFA Fatigue
- Brute Force Attacks
- Password Spray
- Suspicious Inbox Rules
- OAuth Abuse
- Malware Alerts
- Beaconing Activity
- Privilege Escalation
- VPN Anomalies

Skills:
- Incident Triage
- Timeline Analysis
- IOC Enrichment
- User Validation
- Escalation Handling

## Phase 5 — Cloud Security

Learn:
- Azure Fundamentals
- AWS Basics
- Microsoft Entra ID
- Microsoft Defender XDR
- Conditional Access
- Identity Protection

Investigations:
- Risky Users
- Impossible Travel
- Token Theft
- OAuth App Abuse
- Suspicious Sign-ins

---

## Phase 6 — Threat Hunting

Learn:
- IOC Hunting
- Threat Intelligence
- KQL Queries
- SPL Queries
- Process Tree Analysis
- Registry Analysis
- PowerShell Investigations

Hunt For:
- LOLBins
- Encoded PowerShell
- Persistence
- Suspicious Parent-Child Processes
- Command & Control Activity

## Phase 7 — Detection Engineering

Learn:

# SOC Analyst Daily Routine

Typical SOC workflow:
1. Monitor alerts
2. Triage incidents
3. Validate suspicious activity
4. Perform IOC analysis
5. Correlate logs
6. Escalate confirmed threats
7. Document findings
8. Improve detections
